# BSNTools.Web
## Eine Kollektion nützlicher Werkzeuge für das Fach BSN (Betriebssysteme und Netzwerke) als Web-App

### Differenz zu BSNTools
BSN Tools Web ist eine Web-App Variante von BSN Tools, die ursprünglich als GUI-Anwendung entwickelt wurde. Diese Version bietet eine benutzerfreundliche Oberfläche und ist über einen Webbrowser zugänglich, was die Nutzung erleichtert.
Zudem stehen einige Fragen und Probleme im Raum, die in der GUI-Anwendung eventuell rechtzeitig oder auf Dauer gelöst werden konnten (vorallem Performance-Probleme und
die Notwendigkeit von .NET (10) und wie es gebundled wird).

### Verfügbare Werkzeuge
- **Nummernsystemumrechner**: Ein Tool zur Umrechnung zwischen verschiedenen Zahlensystemen (Binär, Dezimal, Hexadezimal, Oktett).
- **IPv4-Netzwerk-Rechner**: Ein Tool zur Berechnung von Subnetzen, Netzmasken, Broadcast-Adressen, und mehr, für IPv4-Adressen.
- **Unit-Converter**: Ein Tool zur Umrechnung verschiedener Einheiten die in BSN relevant sind (z.B. Bits zu Bytes, Zeit, Geschwindigkeit).
- **Subnetting**: Ein Tool zur Planung und Berechnung von Subnetzen.
